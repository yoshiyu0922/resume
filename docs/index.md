## 自己 PR

:::details プロジェクト推進力とコミュニケーション能力 {open}

株式会社 JMDC ではテックリードとしてプロジェクトマネジメントや横断的に技術支援をしてきました。テックリードとして関わったプロジェクトに関して、360 度評価でプロジェクト推進力が高いと評価をいただいたことがあります。また、開発メンバーに技術的な教育を実施する一方で、非エンジニアである企画側との要件整理などコミュニケーションをとりながら開発をしているため、エンジニア・企画・経営層など様々な立場の人と円滑にコミュニケーションを取れると考えています。さらに、開発や運用観点の課題には主体的に取り組み、時には関係者を巻き込んで積極的にリードをして課題解決していきました。パーソルキャリアでも同じ観点を意識して開発をしたことで、上司・同僚・プロダクトオーナーから高い評価をいただきました。

:::

:::details 技術選定とアーキテクチャ設計力 {open}

複数のプロジェクトでテックリードとして技術選定・アーキテクチャ設計を担当してきました。システムの特性や開発メンバーのスキルセットを考慮した適切な技術選定により、開発効率と保守性の両立を実現しています。特に JMDC では、SQL ファーストな bun を採用したことで、複雑な SQL の可読性が向上し、開発効率の向上に貢献しました。また、クリーンアーキテクチャを採用したことで、メンテナンスがしやすいとメンバーから評価を得ることができました。さらに、技術選定・アーキテクチャ設計だけでなく、ガイドラインの作成やメンバーへのレクチャーなども行い、技術的なリードをしてきました。

:::

:::details キャッチアップ力を活かした開発 {open}

実務経験のない技術を採用した案件に参画することが多くありました。Go、Scala、PHP のプロジェクトはそれに該当し、上司・メンバーからキャッチアップの早さを評価していただきました。またプログラミング言語に限らず Terraform、dbt、ory、React、Nuxt などフロントエンドやインフラ関連を数多くキャッチアップをして開発に貢献をしていきました。そのため、新技術に対する抵抗はなく、技術の変化が激しい時代においても、柔軟に対応できる自信があります。

:::
